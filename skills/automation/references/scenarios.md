# 시나리오 가이드

## 시나리오별 필요 도구

| 시나리오 | 필요 도구 | 출력 형태 | 와우 포인트 |
|---------|----------|----------|-----------|
| 주간 업무 리포트 | Jira + Confluence | 위키 페이지 | 내 실제 티켓이 보고서로 변환 |
| 회의록 작성 | Slack + Confluence | 위키 페이지 | 채널 대화가 깔끔한 문서로 |
| 티켓 일괄 생성 | Jira | Jira 티켓 | 한번에 여러 티켓 생성 |
| 프로젝트 현황 요약 | Jira | 화면 표시 | 전체 현황을 한눈에 |
| 프로젝트 현황 PPT | Jira | PPT 파일 | 발표자료를 자동 생성 |
| 티켓 엑셀 정리 | Jira | Excel 파일 | 티켓 목록을 엑셀로 깔끔 정리 |
| Slack→Jira 티켓 변환 | Jira + Slack | Jira 티켓 | Slack 논의를 바로 티켓으로 |
| Jira 현황 Slack 공유 | Jira + Slack | Slack 메시지 | 프로젝트 현황을 채널에 공유 |
| 데이터 리포트 | BISKIT + Confluence | 위키 페이지 | BISKIT 데이터를 자동 분석 |
| BISKIT 데이터 엑셀 | BISKIT | Excel 파일 | 데이터를 엑셀로 분석/공유 |

## 연결 도구 조합별 추천 시나리오

### Jira만 연결된 경우
1. **티켓 일괄 생성** (추천) — 바로 와우 포인트 체험 가능
2. **프로젝트 현황 PPT** — 현황을 발표자료로 자동 생성
3. **티켓 엑셀 정리** — Jira 티켓을 엑셀로 깔끔하게 정리
4. **프로젝트 현황 요약** — 내 프로젝트 현황을 한눈에

### Jira + Confluence
1. **주간 업무 리포트** (강력 추천) — 가장 실용적인 와우 포인트
2. **티켓 일괄 생성**
3. **프로젝트 현황 PPT** — 위키에도, PPT로도 만들 수 있어요
4. **프로젝트 현황 요약**

### Slack + Confluence
1. **회의록 작성** (추천) — 가장 직관적인 와우 포인트
2. **채널 논의 정리** — 주요 결정사항을 위키에 정리
3. 사용자 자유 시나리오

### Slack만 연결된 경우
1. **채널 요약** — Slack 채널의 최근 대화 요약
2. 사용자 자유 시나리오 (Slack 검색 기반)

### Jira + Slack (Confluence 미연결)
1. **Slack→Jira 티켓 변환** (추천) — Slack에서 논의된 할 일을 바로 Jira 티켓으로
2. **Jira 현황 Slack 공유** — 프로젝트 현황을 Slack 채널에 공유
3. **티켓 일괄 생성** — 할 일을 한번에 Jira 티켓으로
4. **프로젝트 현황 PPT** — 현황을 발표자료로 자동 생성
5. **티켓 엑셀 정리** — Jira 티켓을 엑셀로 깔끔하게 정리

> Confluence가 없으므로 결과를 PPT, Excel, Slack 메시지, 화면 표시, 또는 Jira 코멘트로 저장

### Jira + BISKIT (Confluence 미연결)
1. **프로젝트 현황 + 데이터 종합** (추천) — Jira 이슈와 BISKIT 데이터를 한눈에
2. **프로젝트 현황 PPT** — 종합 데이터를 발표자료로
3. **티켓 엑셀 정리** — 티켓 + BISKIT 데이터를 엑셀로
4. **티켓 일괄 생성** — 할 일을 한번에 Jira 티켓으로

> **참고**: Confluence가 없으면 결과를 PPT, Excel, 화면 표시, 또는 Jira 코멘트로 저장

### BISKIT 포함 (Confluence 연결됨)
1. 위의 시나리오 + **데이터 리포트** 추가 제안 — BISKIT 데이터를 위키에 자동 분석
2. **BISKIT 데이터 엑셀** — 데이터를 엑셀로 내보내기

### 전체 연결 (Jira + Confluence + Slack + BISKIT)
1. **주간 업무 리포트** (첫 번째 추천)
2. **회의록 작성** (두 번째 추천)
3. **티켓 일괄 생성** (세 번째 추천)
4. **프로젝트 현황 PPT** — 발표가 필요하면 추천

## PPT/Excel 시나리오 상세

### PPT 시나리오 예시
- **프로젝트 현황 발표**: Jira 이슈 현황 → 표지/요약/상세/리스크 슬라이드
- **스프린트 리뷰 발표**: 스프린트 완료/미완료 → 성과 슬라이드
- **BISKIT 데이터 발표**: BISKIT 지표 → 차트/인사이트 슬라이드
- **주간보고 PPT**: 주간 업무 → 요약 슬라이드 (위키 대신)

### Excel 시나리오 예시
- **티켓 엑셀 정리**: Jira 전체 티켓 → 필터/서식 적용된 엑셀
- **BISKIT 데이터 엑셀**: BISKIT 데이터셋 → 시트별 정리된 엑셀
- **스프린트 이력 엑셀**: 여러 스프린트 이슈 → 비교 분석 엑셀
- **일감 현황 엑셀**: 팀원별 이슈 배분 → 현황 파악용 엑셀

## 사용자 자유 입력 처리

사용자가 목록에 없는 업무를 말한 경우:
1. 연결된 도구로 가능한지 판단
2. 가능하면 즉석에서 실행
3. 불가능하면 필요한 도구를 안내하고 가능한 대안 제시
4. **출력 형태를 확인**: 위키 / PPT / Excel / 화면 중 선호하는 형태
